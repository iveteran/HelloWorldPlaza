Commands:
fc-list
fc-match <keyword>
fc-cache -f -v

User's font directory:
~/.local/share/fonts

Install font:
cp your-fonts/<font-name>.ttf ~/.local/share/fonts
fc-cache -f -v

Uninstall font:
rm ~/.local/share/fonts/<font-name>.ttf
fc-cache
