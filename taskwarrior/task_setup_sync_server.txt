0) Install(Debian)
  $ sudo apt install taskwarrior

1) Check version, it must greater than 2.3.0
  $ task --version

2) Download keys that generated by Freecinc.com
  Go to the site to download three files: *.key.pem, *.cert.pem and *.ca.pem,
  And put them in ~/.task

3) Run follow commands to generate config
  $ task config taskd.server freecinc.com:53589
  $ task config taskd.key ~/.task/freecinc_6bfa58b6.key.pem
  $ task config taskd.certificate ~/.task/freecinc_6bfa58b6.cert.pem
  $ task config taskd.ca ~/.task/freecinc_6bfa58b6.ca.pem
  $ task config taskd.credentials -- 'FreeCinc/freecinc_6bfa58b6/909dae80-aad8-4df1-ab3d-33139dbad3c0'

4) Verify config
  $ task diagnostics
  MAKE SURE THAT IN THE OUTPUT CA, Cert, AND Key ARE ALL "READABLE".
  ALSO ENSURE THAT YOU HAVE A VALUE FOR Server AND A VALUE FOR Creds.

5) Sync
  $ task sync init

6) Setup on another machine (OPTIONAL):
  IF YOU WANT TO SET UP ANOTHER COMPUTER TO SYNC WITH THE SAME ACCOUNT,
  COPY ALL KEYS THAT YOUR DOWNLOADED AT STEP 2 TO YOUR OTHER COMPUTER,
  AND COMPLETE STEPS 0-5.


Ref: https://taskwarrior.org/
