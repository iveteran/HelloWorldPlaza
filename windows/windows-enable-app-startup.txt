How to launch apps automatically during startup on Windows 10/11

Ref: https://www.windowscentral.com/how-launch-apps-automatically-during-login-windows-11


How to configure app automatic start with Startup folder
1) Run following command(Win + R): shell:startup, To open a folder: C:\Users\ivete\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
2) Click the New menu and select the Shortcut option, To create a shortcut to the program that your want auto start at Windows started.
