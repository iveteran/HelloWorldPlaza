Enable DNSSEC on Linux

refer:
  https://wiki.debian.org/DNSSEC

1. With DNSCrypt-proxy
sudo dnf install dnscrypt-proxy
sudo systemctl start dnscrypt-proxy
sudo systemctl status dnscrypt-proxy
Edit /etc/resolv.conf:
nameserver 127.0.0.1

2. With Unbound, dnssec-trigger
1) Installing Unbound
apt-get install unbound

2) Installing dnssec-trigger
apt-get install dnssec-trigger

3) Modify your /etc/resolv.conf to use loopback interface as nameserver.
nameserver 127.0.0.1

3. verifing by using command: host yahoo.com (OR dig yahoo.com):
1) unused dnscrypt: tcpdump -vv -x -X -s 1500 -i eth0 udp port 53   -> will captures plain packets of DNS query
2) used dnscrypt:   tcpdump -vv -x -X -s 1500 -i eth0 udp port 443  -> will captures encrypted packets of DNS query

4). test
dig +dnssec iveteran.me
